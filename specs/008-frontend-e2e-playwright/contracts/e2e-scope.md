# Contract: E2E Scope

## Fluxos obrigatórios
- cadastro de usuário
- login de usuário existente
- criação de tarefa
- mudança de status na tabela
- mudança de status no kanban
- arquivamento e desarquivamento
- exclusão definitiva de tarefa arquivada

## Requisitos operacionais
- backend e frontend iniciam automaticamente para a suíte
- browser usado pela suíte é o Chromium do Playwright
- dados de teste são únicos por execução
